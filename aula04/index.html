Tags semânticas da estrutura do HTML


O html lançou tags semânticas que são específicas para serem usadas para objetivos bem definidos e isso, facilita e muito, o desenvolvimento em diversos aspectos.


Referência: Douglas Novato

Após ler o texto acima, provavelmente você pensou:
Como assim?! depois de ter aprendido a criar meus códigos apenas com divs para separar os conteúdos vou ter que alterar tudo para estas tais tags "semânticas"?
A resposta é: SIMMMM!
Bem, se seu código é apenas uma base de um projeto maior, isso deve ser feito o quanto antes para diminuir o trabalho posterior ao término do projeto.
Já imaginou ter quer alterar várias tags de um projeto pronto só por causa de novas tags implementadas?
Cansativo só de pensar concorda? então, entenda-as e as utilize.
Entre a tag div e as tags semânticas não há essa "rivalidade" ilustrada na imagem que vimos acima, não há uma melhor que a outra, ou a "correta".
Não se deve descartar a utilização de nenhuma das duas, elas se complementam, e, trabalhando em conjunto melhoram o SEO dos sites.
